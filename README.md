# Smartphone-Addiction-Analysis
A Data visualization analysis of how smartphone usages impact on work or study and its impact on addiction level and stress level.

The original raw dataset is from this website: https://www.kaggle.com/datasets/jayjoshi37/smartphone-usage-and-addiction-prediction

I clean the dataset using Excel and analysis it using Power BI and Tableau. This analysis focus on smartphone screen time impact on mental health of Gen Z and Millennials age group. 

This project can be used by downloading the files in the repository. If you have Excel, Power BI and Tableau app, it can be opened by double clicking the files.
